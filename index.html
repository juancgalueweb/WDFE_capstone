A little bit about me

I am a Venezuelan guy living in Chile, and an electrical engineer with more than 14 years in the industry, specifically in power generation and distribution, but since the pandemic I started to realize that I wanted to do something different with my life. I found programming a fascinated subject in the high school, but somehow I lost the path and now I want to reconnect with my passion for programming. I want to become a Front-end Web Developer and master JS and then learn React. Later I want to explore the Back-end.

Technologies used to build the photo gallery

All the information displayed in the web page was generated using HTML 5 tags and syntax.

CSS is a great tool to stylish your page. In this case I have also used CSS3 to override some Bootstrap initial setups, and to add styles to FontAwesome icons.

I used JavaScript to implement a nice touch to the pictures in the gallery, using an existing plugin. I also used it to manage the alerts when there are invalid fields in the form and when it is successfully “submitted”.

I decided to write the code from the scratch with the help of a framework, in this case, Bootstrap 5. I used it to arrange the content in the layout, to have a responsive layout, to use buttons, a contact form, a navigation bar, a footer, and so on.
